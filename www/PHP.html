Baidu Statistics

Platform: Windows Operation System
Language: PHP, HTML, CSS, Javascript, Bootstrap
Development Tool: WAMPSERVER, Easy Eclipse for PHP
Database: SQL Server 2008
Functions:
  1. User Management
  2. Download Baidu Statistics Report
  3. Download Baidu NetAlliance Report
  4. Download Baidu Search Report
  5. Display the downloaded report
  6. Read data from CSV file and insert into SQL Server database

Features:

1. User Management
  • Retrive users from SQL Server database and display
  • Add a new user
  • Update the profile of an existing user
  • Delete an user
  • Page Navigation
2. Download Baidu Statistics Report
  • Connect to Baidu server, set up a session between client and server
  • Sent a request(JSON format which contains username, password, session id, date) to Baidu server
  • Query the generating status of report periodically
  • Retrieve a link from Baidu server and download report
  • Save the csv report to local disk
  • Read data from csv file, display the report and insert data into database
3. Download Baidu NetAlliance Report
  • Connect to Baidu server, set up a session between client and server
  • Sent a request(JSON format which contains username, password, session id, start date, end date) to Baidu server
  • Query the generating status of report every five seconds, fail after three times querying
  • Retrieve a link from Baidu server and download report
  • Save the csv report to local disk
  • Read data from csv file, display the report and insert data into database
4. Download Baidu Search Report
  • Connect to Baidu server, set up a session between client and server
  • Sent a request(JSON format which contains username, password, session id, start date, end date) to Baidu server
  • Query the generating status of report every five seconds, fail after three times querying
  • Retrieve a link from Baidu server and download report
  • Save the csv report to local disk
  • Read data from csv file, display the report and insert data into database
5. Demonstrate Report through Chart
  • Select year, account and chart type
  • Demonstrate Report through bar chart
  • Demonstrate Report through line chart
  • Demonstrate Report through pie chart
