Architecture of Chatting System

Component Description Output Implement
WCF Service Windows Communication Foundation (WCF) is a framework for building service-oriented applications. Using WCF, you can send data as asynchronous messages from one service endpoint to another. The interfaces defined in WCF service can be called on any endpoint(client application). Service.dll Declare and define interfaces in WCF service
Server Application A WCF service can not run independently. It must parasitize on a host application. In this system, the server application is the host of WCF service. Host.exe start WCF service on host application
Client Application Import WCF service, call interfaces defined in WCF service to communicate with server application or other endpoint(client application). Client.exe import WCF service in client application
