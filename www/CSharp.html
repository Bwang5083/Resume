Basic Information:

Name: Book Reader
Platform: Windows Operation System
Language: C#, WPF, MVVM, SQL, HTML, CSS
Development Tool: Visual Studio 2013
Database: SQL Server 2014
Functions:
  1. Main interface
  2. Reading interface
  3. Resource dictionary library
  4. Using SQL Server to store all information
  5. Applying MVVM design pattern

Functionalities:

1. Main Interface
  • Menu bar
  • Tool bar
  • Tree view
  • Grid view(book list or magazine list)
  • Status bar(display rolling advertisement)
2. Reading interface
  • Menu bar
  • Tool bar
    • Add bookmark
    Delete bookmark
    Page navigation
    Zoom in & Zoom out
  • Content view
  • Book catalogue view
  • Bookmark view
    • Ouline all bookmarks
3. Resource dictionary library
  • Develop a general resource library which can be imported to other projects
4. Using SQL Server to store information
  • Three layered architecture: data access layer(DBOperation), business logic layer(model), presentation layer(xaml)
  • Tables in SQL Server database
    • Advertisement: store advertisement content
    BookMark: store bookmark information
    catalogue: store catalogue of all books
    EMagazine: store electronic magazions' information
    LocalShelf: store the information of all books
    TreeView: store treeview nodes' information
5. Applying MVVM design pattern
  • Define model
    • TreeView model example
  • Bind model with a control's or a window's DataContext
    • Bind model with TreeView control example
  • Bind a model property with interface
    • Bind property DigitalResources with TreeView example
