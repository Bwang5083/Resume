Database: SQL Server 2012

  7. User management
  8. Localization(English, French, Chinese)

System Architecture

Model:
Encapsulates application state; Responds to state query and state update; Notify views of changes.
View:
Renders the models; Requests updates from models; Allow controller to select view.
Controller:
Defines application behavior; Map user action to model updates; Selects view for response.
