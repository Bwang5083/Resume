Basic Information:

Project Name: XML File Editor
Platform: Windows Operation System
Language: C++, MFC, XML API
Middle Ware: msxml4.dll
Development Tool: Visual C++ 6.0
XML File: PamSysLofSettings.price_list.xml
Functionalities:
  1. Display the content of XML file in window
  2. Add a new node to XML tree
  3. Update a node of XML tree
  4. Remove a node from XML tree

This project is designed and developed for a third part company ESI. The company requested us to develop a tool through which they can edit a XML file. In other words, the customer wants a tool to add, delete and update a XML file. In the past, they do this job mannually which is time-consuming and boring. This is a general tool which can be used to edit any XML file.

Architecture of XML File Editor

Component Description Output
Main Project
  1. Re-encapsulate(overload) XML API defined in middle ware, add some customized features
  2. Add a new node to XML tree, then save to XML file
  3. Update the information of a XML tree node, then save to XML file
  4. Remove a node from XML tree as well as XML file
 ModifyXML.exe
Middle Ware Import this middle ware to invoke XML API msxml4.dll

Project Name: Quotation Generator
Platform: Windows Operation System
Language: C++, MFC, XML API, Microsoft Office API
Middle Ware: msxml4.dll
Development Tool: Visual C++ 6.0
XML File: PamSysLofSettings.price_list.xml
Functionalities:
  1. Extract data from XML file and display in window
  2. Form a product price list
  3. Generate quotation report(example report)

This project is designed and developed for a third part company ESI. The company provides us with a XML file which contains all its products' price. The requirement is to extract the XML file and display the products' price on interface for users to select. User can create a dynamic product price list in system, then generate a Excel version quotation report via this system.

Basic Information:

Project Name: Xian Aircraft Project
Platform: Windows Operation System
Language: C++, MFC, STL, GUI
Middle Ware: MSO.DLL, MSWORD.OLB
Development Tool: Visual C++ 6.0
Functionalities:
  1. Verify aircraft design parameters
  2. Verify horizontal empennage design
  3. Verify vertical empennage design
  4. Generate word version report

This project is designed and developed for AVIC XI'AN AIRCRAFT INDUSTRY(GROUP) COMPANY LTD. Two doctors of Northwestern Polytechnical University are responsible to collect requirements and complete the requirement document. I assisted them to do software development relevant work.

  1. Verify aircraft design parameters:

    2. Develop an executable(HVModel.exe) with VC++ to verify the aircraft design parameters.

    Input: a txt file which contains all the design parameters (design parameter input file).

    Output: a parameter check list, tick means the parameter is correct, cross means the parameter is wrong.

  2. Verify horizontal empennage design:

    3. Develop an executable(Tail.exe) with VC++ to verify the horizontal empennage design parameters.

    Input: a txt file which contains all the design parameters (horizontal empennage design input file).

    Output: a line chart in which each line represents a specific parameter.

  3. Verify vertical empennage design:

    4. Develop an executable(VTail.exe) with VC++ to verify the vertical empennage design parameters.

    Input: a txt file which contains all the design parameters (vertical empennage design input file).

    Output: a line chart in which each line represents a specific parameter.

  4. Generate word version report:

    5. Develop an executable(txtword.exe) with VC++ to generate a word version report.

    Input: a txt file which contains all parameters.

    Output: a word report (sample report).
