 Bus.pmc

System Skin

  1. Search special food of a selected city

 special.pmc
Train time table
  1. Search available trains according to departure station and destination
  2. Search available trains according to station name. In other words, which trains pass this railway station
  3. Search available trains according to train number
  4. Input prompt for departure station, destination and station name
  5. Find out all available trains and display in HTML file (available train routes)
  6. Look up all stations of a selected train route (stations of a train route)
 TrainTable.pmc

  4. Locate the scenic spot on map (locating)
 travel.pmc
